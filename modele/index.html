Qui sommes nous ?

Cette plateforme web à été mise à diposition de tout étudiant souhaitant emprunter du matériel informatique et/ou électronique. Elle est mise en place depuis 19 juin 2022 !

Application web réalisée grâce notre passion du code informatique et de l'algorithmique ;=)

Tout en adoptant le design parttern MVC (Modèle/Vue/Contrôle), L'application web à été développer en méthode agile par les developeur full-Stack LACHHAB Adrien : ans et SCHIAVO Corentin !

Voici la liste de toutes les outils et technologies utilisées lors de la création de notre plateforme

Front-end !

Pour l'emsemble des pages ayant été développés, nous avons utilisées les technologies courantes que sont le HTML5, le CSS3, le JavaScript ainsi qu'un framework CSS développé par un ingénieur de twitter qui se nomme Bootstrap

Back-end !

Pour l'emsemble de la programmation côté serveur nous nous sommes servis de PHP. C'est un langage qui est de moins en moins populaire en 2022 mais encore très utilisé par la communauté de développeurs webs. Par ailleurs la plupart des applications webs hébergés sur l'internet mondial sont écris en PHP pour le côté back-end

Outils de gestion de projet !

Plusieurs outils nous ont été indispensables pour le developpement de notre site. Il s'agit d'une base de données relationnelle (MySQL), l'utilisation de Git bash/github pour la sauvegarde par partie du code ainsi qu'un outil de gestion de projet pour repérer les différentes tâches confiés (Trello) !
